# Pratikum-6

1. tambah()

Menambahkan data mahasiswa baru.
Pengguna mengisi nama, nilai tugas, UTS, dan UAS.
Program kemudian menghitung nilai akhir berdasarkan rumus:

Tugas: 30%

UTS: 35%

UAS: 35%

2. tampilkan()

Menampilkan seluruh data mahasiswa dalam bentuk tabel sederhana.
Jika belum ada data, program memberi pesan bahwa data masih kosong.

3. hapus(nama)

Menghapus data mahasiswa berdasarkan nama.
Jika nama tidak ditemukan, program akan menampilkan pesan peringatan.

4. ubah(nama)

Mengubah data mahasiswa yang sudah ada.
Nilai baru akan dihitung ulang untuk mendapatkan nilai akhirnya.

Cara Menggunakan Program

Jalankan program dengan perintah:

    python main.py


Pilih menu sesuai kebutuhan:

1 → Tambah data

2 → Tampilkan data

3 → Hapus data

4 → Ubah data

5 → Keluar dari program

Ikuti instruksi yang ditampilkan pada layar.

Flowchart Program

Flowchart menggambarkan alur utama program, mulai dari menampilkan menu, memilih opsi, menjalankan fungsi tertentu, hingga kembali lagi ke menu utama atau keluar dari program.

(Di bagian repository, flowchart dapat disertakan dalam bentuk gambar.)

Tujuan Pembuatan

Program ini dibuat sebagai bagian dari praktikum pemrograman Python dengan tujuan:

Menerapkan penggunaan fungsi dalam program

Melatih pengelolaan data menggunakan dictionary

Memahami konsep input, output, dan perhitungan nilai

Membuat program sederhana yang interaktif
